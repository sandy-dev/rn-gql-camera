### Example image gallery app to demonstrate GraphQl with React Native 

#### Stack
- ReactNative
- GraphQL
- RN Camera Fetaures
- MaterialUI

#### Installation
- git clone rn-gql-camera
- cd rn-gql-camera
- yarn install
- yarn dev
